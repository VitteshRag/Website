Hooked – In Depth Project Details

  • Project Type: Personal Project
  • Team Size: 1
  • Duration: 3 weeks
  • My Role: Game Designer, Developer

Project Summary

"Hooked" is a casual game with core mechanics like casting, catching, inventory, and shopkeeping. The game is designed for satisfying loop-based gameplay and UI-driven interaction, offering players a relaxing yet engaging experience.

What I Did

  • Designed and implemented the fishing and shopkeeping mechanics.
  • Created the inventory and progression systems.
  • Developed the user interface and overall game flow.

Key Features

  • Intuitive casting and catching mechanics.
  • Inventory management and upgrades.
  • Shop system for selling and buying items.

Challenges & Solutions

The main challenge was balancing the gameplay loop to keep it rewarding without becoming repetitive. I iterated on the reward structure and UI feedback to maintain player engagement.
